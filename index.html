// MiroFish Lab

DAO Governance
Agent Simulations

MiroFish is an agent simulation framework for modelling how DAOs make decisions — before those decisions go on-chain and become irreversible.

I forked MiroFish, extended it with custom AI personas and scenario assumptions for protocol and DAO governance contexts, and integrated an ElevenLabs voice agent as one of the simulated stakeholders.

Each simulation models a realistic governance scenario — a treasury vote, a protocol upgrade, an incentive structure change — with distinct stakeholder personas that argue, deliberate, and vote. The output is a structured writeup of outcomes, minority positions, and risk flags.

MiroFish fork ElevenLabs voice LLM personas DAO governance Protocol simulations

How a simulation runs

1
Define the governance scenario — treasury allocation, parameter change, protocol upgrade, or incentive design.
2
Assign stakeholder personas — large token holders, retail delegators, core devs, ecosystem partners, and adversarial actors.
3
Run deliberation rounds — agents debate, form coalitions, and update positions based on arguments.
4
Surface outcomes — likely vote result, minority dissent, attack vectors, and risk flags for the protocol.
5
Voice replay — ElevenLabs agent narrates the deliberation as a listenable walkthrough.
